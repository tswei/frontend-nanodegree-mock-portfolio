# Portfolio Mockup to HTML P1 Front-end Nanodegree

## Project Overview

Portfolio Mockup to HTML is a Udacity Frontend Nanodegree project teaching the basics of HTML and CSS. The goal of the completed project is to translate a pdf mock into a functionally styled webpage.

## Install

For the latest version, clone this [repository](https://github.com/tswei/frontend-nanodegree-mock-portfolio)

To install dependencies, [Node.js](https://docs.npmjs.com/getting-started/installing-node), [Grunt](http://gruntjs.com/getting-started), and [Bower](http://bower.io/#install-bower) are required. Instructions on installing these can be found by following the links. They should all be installed globally.

The Grunt CLI and Bower can be installed via npm:

`npm install -g grunt-cli`

`npm install -g bower`

To install the dependencies, run:

`npm install`

`bower install`

## Instructions to Run

After following the instructions for installation, open the main html file, index.html, in your browser using the URL file:///C:/PATH/TO/ROOT/assets/index.html.

A live view of this project will be available soon!
